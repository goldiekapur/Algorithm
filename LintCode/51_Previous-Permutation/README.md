51. Previous Permutation
中文English
Given a list of integers, which denote a permutation.

Find the previous permutation in ascending order.

Example
Example 1:

Input:[1]
Output:[1]
Example 2:

Input:[1,3,2,3]
Output:[1,2,3,3]
Example 3:

Input:[1,2,3,4]
Output:[4,3,2,1]

Notice
The list may contains duplicate integers.
