1254. Sum of Left Leaves

Find the sum of all left leaves in a given binary tree.

Example
Example 1

Input：
{3,9,20,#,#,15,7}
Output：24

Explanation：There are two left leaves in the binary tree, with values 9 and 15 respectively. Return 24.
    3
   / \
  9  20
    /  \
   15   7
Example 2:

Input:
{1,#,2,#,3}
Output:
0

Explanatinon:
1
  \
    2
      \
       3
