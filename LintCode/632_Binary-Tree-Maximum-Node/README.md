632. Binary Tree Maximum Node
中文English
Find the maximum node in a binary tree, return the node.

Example
Example 1:

Input:
{1,-5,3,1,2,-4,-5}
Output: 3
Explanation:
The tree look like this:
     1
   /   \
 -5     3
 / \   /  \
1   2 -4  -5
Example 2

Input:
{10,-5,2,0,3,-4,-5}
Output: 10
Explanation:
The tree look like this:
     10
   /   \
 -5     2
 / \   /  \
0   3 -4  -5
